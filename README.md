# 43389C Box Bot

A VEX robotics competition robot code built by the Canadian Academy Robotics Team.

## Hardware Configuration

### Motors
- **Left Drive Motor** - Port 1 (18:1 gear ratio)
- **Right Drive Motor** - Port 2 (18:1 gear ratio, reversed)
- **Intake Motor** - Port 3 (18:1 gear ratio)

### Sensors
- **Optical Sensor** - Port 5 (color detection)
- **GPS Sensor** - Port 6 (position tracking)

### Robot Dimensions
- Wheelbase: 320mm
- Track width: 280mm
- Wheel diameter: 319.19mm

## Controls

### Driver Control
- **Left Joystick (Axis 3)** - Forward/Backward
- **Right Joystick (Axis 1)** - Turning
- **R1 Button** - Run intake forward
- **R2 Button** - Run intake reverse

## Competition Modes

### Autonomous
The autonomous code

### Driver Control
Full manual control of the robot with driving and intake controls.
